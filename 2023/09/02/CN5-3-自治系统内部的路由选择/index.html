CN5.3 自治系统内部的路由选择

Author: Simba Date: September 02, 2023  11:07:01 Category: Computer Network Words: 347 Time: 1min

Section 5 网络层：控制平面

中科大郑烇、杨坚老师《计算机网络》笔记

5.3 自治系统内部的路由选择

  • Routing Information Protocol (RIP)
    • DV 算法
    • 每条链路cost=1，max hops = 15
    • 每隔30秒和邻居交换通告（dest-subnet｜ hops-to-dest），或者收到请求的时候
    • 每个通告包括最多25个目标子网的可达信息
    • 180(6个周期)没有收到链路通告，则链路失效
    • 水平分裂阻止坏消息形成环路
  • Open Shortest Path First (OSPF)
    • LS算法，在一个AS内部泛洪，算汇集树，算路由表
    • 路由通告，携带信息：版本、ttl、邻居、到邻居的代价
    • 安全性：所有报文经过认证，防止恶意攻击
    • 允许多个代价相同的路径存在，可以做负载均衡（RIP只保留一条）
    • 允许多重代价矩阵，支持按照不同的代价计算最优路径
    • Multicast OSPF
    • 大型网络中支持层次化OSPF
      • 2个级别的层次性：本地/骨干；链路状态通告仅在本地area内进行
      • 区域边界路由器：汇总自己区域内网络的距离，向其他区域的边界路由器通告
      • 骨干路由器：仅在骨干区域内，运行OSPF路由
      • 边界路由器：连接其他的AS
  • IS-IS与OSPF几乎相同
